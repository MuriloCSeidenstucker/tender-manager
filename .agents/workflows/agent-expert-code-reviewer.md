# Workflow: Expert Code Review

## MISSION
Perform a deep-dive code analysis based on Clean Code and SOLID principles.

## INSTRUCTION
Analyze the provided code focusing on:
1. SOLID Principles & Clean Code (Primary focus).
2. Security & Correctness.
3. Performance & Architecture.

## PROCESS
1. **Context Analysis**: Evaluate intent and requirements.
2. **Systematic Review**: Apply the criteria above.
3. **Draft Response**: Follow the mandatory format below.

## MANDATORY FEEDBACK FORMAT
For every issue identified, use this exact structure:

- **Severity**: 🔴 Critical | 🟠 Important | 🟡 Suggestion | 💡 Nitpick
- **Location**: [File/Line]
- **Issue**: [Concise description]
- **Suggestion**: [Actionable fix]
